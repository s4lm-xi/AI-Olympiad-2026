## Repository Overview

This repository contains the official implementation files for AI Olympiad 2026.

The repository is organized into three main folders covering two AI tracks:

1. Track 1: AI Development
2. Track 2: AI Applications

Some folders are still work in progress and will be updated during development.

Each track folder contains the required code, datasets, model training files, and evaluation scripts needed to complete the challenge.

## Tracks Overview

## Track 1: AI Development

### Build LLM from Ground Up

The goal of this challenge is to create a language model from the ground up.

Participants will build the main components of an LLM including:

Tokenizer creation

Dataset preparation

Model architecture design

Model training

Text generation and evaluation

The objective is to understand how large language models work by implementing the main pipeline from data collection to a working text generation model.

## Track 2: AI Applications

## AI Agentic for Video Analysis

The goal of this challenge is to build an agentic AI solution that analyzes video footage and detects potential bullying indicators.

The system should use AI agents to process video information, understand events, and provide meaningful analysis.

Possible tasks include:

Video processing

Event understanding

Detection of suspicious activities

Generating reports or explanations

## AI Agentic Reading Old Text

The goal of this challenge is to create an agentic AI solution that reads and interprets old or difficult text from scanned documents, handwritten records, or historical archives.

The system should help extract and understand information from challenging documents.

Possible tasks include:

Image preprocessing

OCR extraction

Text correction

Historical text understanding

Information retrieval and explanation
